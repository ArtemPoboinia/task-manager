# Task Manager

Современный task manager, созданный на React + TypeScript.

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- LocalStorage

## Features

- Создание задач
- Удаление задач
- Отметка выполнения
- Фильтрация
- Сохранение после перезагрузки
